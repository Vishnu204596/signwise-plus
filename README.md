# SignWise+ 🤟  
Real-Time Sign Language Recognition using Computer Vision & Deep Learning

## 🚀 Live Demo
👉 https://signwise-real-time-gesture-ai.onrender.com

## 📌 Overview
SignWise+ is a real-time sign language recognition system that converts hand gestures into text and speech using computer vision and deep learning techniques.

## 🧠 Features
- Real-time gesture recognition
- Multiple models (YOLOv8, ResNet50, EfficientNet, MLP)
- Dynamic hand tracking (MediaPipe)
- Offline Text-to-Speech
- GUI-based interaction
- Custom dataset (1000+ images)

## 🏗️ Project Structure

Training Scripts/ → Model training notebooks
Inference scripts/ → Real-time application
runs/ → Model outputs & results
Sample Dataset Images/ → Dataset samples


## ⚙️ Tech Stack
- Python
- OpenCV
- MediaPipe
- PyTorch
- TensorFlow/Keras
- Ultralytics YOLOv8
- Flask (for deployment)
